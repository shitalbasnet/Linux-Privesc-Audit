# Linux-Privesc-Audit
Automated Linux Privilege Escalation Enumeration and Security Auditing Toolkit developed using Python and Bash.

## Project Overview
The Linux Privilege Escalation Automation Toolkit is a cybersecurity auditing tool developed using Python and Bash on Kali Linux. The toolkit automates the identification of common Linux privilege escalation vectors and security misconfigurations. It helps security professionals and students understand how attackers perform privilege escalation and how defenders can identify and mitigate these risks.

## Features

The toolkit performs the following security checks:

System Information Gathering
SUID/SGID Binary Enumeration
Weak File Permission Analysis
Cron Job Enumeration
Sudo Privilege Analysis
System Service Enumeration
Linux Capability Scanning
Kernel Information Collection

## Technologies Used
Python 3
Bash Scripting
Kali Linux
Linux Command Line Utilities
